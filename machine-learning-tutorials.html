Setting up a Deep Learning Environment in Python 2.7

Windows: (install Anaconda as the base package)
1. open your command prompt.
2. Type: pip install mingw
3. Type: pip install libpython
4. Type: conda install theano
5. Type: conda install keras

In case, theano already exists, type: pip install theano -- upgrade

The Keras library gives, very well documented and recent algorithms for implementing neural nets and deep learners.

Refer to: http://keras.io/
